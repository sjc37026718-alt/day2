# Brand Analytics — 브랜드 매출 분석 대시보드

AI(Claude) 바이브코딩으로 만든 브랜드 입점현황 · 매출실적 통합 분석 대시보드입니다.

## 무엇을 만들었나

엑셀 파일 2개(입점현황 + 매출실적)를 업로드하면, 자동으로 병합하여
KPI 요약 · 브랜드별 매출 · 채널별 분석 · 카테고리 히트맵까지 한눈에 보여주는 대시보드입니다.

## 주요 기능

- 엑셀 파일 업로드 → 자동 병합 (공통 키 자동 감지)
- KPI 카드: 총매출, 주문건수, 순이익, 브랜드 수
- 브랜드별 매출/순이익 TOP 12 차트
- 채널별 매출 비중, 주문건수, 반품률, 재구매율 분석
- 카테고리 × 브랜드 히트맵
- 실시간 서울 날씨 · 환율 정보 (Open-Meteo, ExchangeRate API)
- CSV 다운로드 기능

## 활용 도구

| 도구 | 역할 |
|------|------|
| Claude | 전체 코드 작성 (바이브코딩) |
| Streamlit | 웹 대시보드 프레임워크 |
| Plotly | 인터랙티브 차트 시각화 |
| Pandas | 데이터 병합 및 분석 |

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 배운 점

- 마케터가 직접 데이터 분석 도구를 만들면 엑셀 수작업 없이 의사결정 속도를 높일 수 있음을 확인
- 무료 API(날씨, 환율)를 붙이는 것만으로 실제 서비스처럼 느껴지는 완성도를 만들 수 있음
